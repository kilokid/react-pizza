## Как использовать

Клонирование репозитория:\
`git clone https://github.com/kilokid/react-pizza.git`

Установка зависимостей:\
`npm install`

Сборка проекта в режиме разработки и запуск локального сервера:\
`npm start`

Сборка в продакшн:\
`npm run build`

### Обязательно требуется:
Запуск json сервера:\
`npm run dev`
